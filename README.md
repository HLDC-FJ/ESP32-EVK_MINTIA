# ESP32 EVK for MINTIA
MINTIA (清涼菓子) ケースを使用し、ESP32 EVK作成を行う。

* ESP32
* Acconeer XM125 (60GHz Radar)
* IR Sensor Array

MINTIAケースを活かし素敵な EVK を作る

このプロジェクトは、ソフトウェアとハードウェアの両方を含みます。
以下のコンポーネントは MIT ライセンスの下で公開されています。

- ソフトウェア（コード、スクリプトなど）
- ハードウェア設計（回路図、基板データ、CADファイルなど）
